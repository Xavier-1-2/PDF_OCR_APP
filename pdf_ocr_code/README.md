# PDF Watcher — Setup Instructions

> Tesseract is included in the project — no separate installation needed.

---

## Requirements

- Python 3.8+

---

## 1. Install Python Dependencies

Open a terminal in the project folder and run:

```bash
pip install -r requirements.txt
```

---

## 2. Set Up the Batch File

1. Open `Run PDF Watcher.bat` in a text editor
2. Replace `C:\Path\To\Project` with the actual path to the project folder on your PC
3. Save the file
4. Move it to your Desktop

From now on, just double-click the batch file to launch the app.
