# PDF Watcher — Setup Instructions

> Tesseract is included in the project — no separate installation needed.

---

## Option A: Run the Executable

No setup required. Just double-click:

```
ocrmain.exe
```

---

## Option B: Run from Script

### Requirements
- Python 3.8+

### 1. Install Python Dependencies

Open a terminal in the project folder and run:

```bash
pip install -r requirements.txt
```

### 2. Run the App

```bash
python pdf_watcher.py
```
